## tag

```shell
watson tag [action] [arguments] [options]
```

Perform actions on tags, such as

- Create a tag
- Remove a tag
- List tags
- Rename a tag

#### Actions

| Action | Description  |
|--------|--------------|
| add    | Create a tag |
| remove | Remove a tag |
| list   | List tags    |
| rename | Rename a tag |

#### Examples

```shell
watson tag add pizza
watson tag remove 02af5e
watson tag list
watson tag rename 02af5e burger
```
