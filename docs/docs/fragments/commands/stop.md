## stop

```shell
watson stop [options]
```

Stop the currently running frame.

#### Options

| Option     | Description                           | Example                   |
|------------|---------------------------------------|---------------------------|
| `-a, --at` | Stop the frame at the specified time. | `--at "2025-01-01 14:45"` |

#### Examples

``shell
watson stop
watson stop --at "2025-01-01 14:45"

```
